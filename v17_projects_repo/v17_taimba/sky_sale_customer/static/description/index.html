Sale Customer

This will allow you to display only customers on the sales orders.

Currently it shows all partners including suppliers as well. But using this module only customers will be shown.

Sales > Orders > Customers
























Customer Form

"Is Customer?" is the field which can be set manually by a user to make a partner a customer. Also, when created from the Customers menu, it will be by default a customer.









**This module is compatible for both the vesrion enterprise and community.
